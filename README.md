# Phototherapy_Timer

Smart replacement timer / controller for a **SolRx E-Series** medium-frame master phototherapy unit (model family includes **E760M-UVBNB**).

The stock timer circuit on the device failed (lamp no longer lights on schedule). This project explores a **UserHackable** controller that:

- Restores reliable timed treatment sessions
- Adds smarter control (schedules, safety limits, clearer status)
- Adds reporting / logging for sessions and device health

## Hardware context

- Device: [SolRx E-Series Master (medium frame)](https://solarcsystems.com/product/medium-frame-solrx-e-series-master/)
- Manufacturer timer replacement is an option; this repo is for a custom controller with extended features
